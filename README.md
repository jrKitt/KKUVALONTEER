
## Getting Started
1. Clone the repository.
```bash
  git clone https://github.com/jrKitt/KKUVALONTEER.git
  cd KKUVALONTEER
```
2. Setup laravel
```bash
  npm install
  composer install
  cp .env.example .env
  php artisan key:generate
```
    
3. Running (npm and php together)
```bash
  npm run dev
  php artisan serve
```
